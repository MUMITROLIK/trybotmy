# Category Analysis System

## Что сделано

Добавлена система сбора данных с категориями сигналов для анализа эффективности.

### 1. Новые поля в БД

Добавлены колонки во все таблицы (signals, taken_trades, ml_features):
- `signal_category` - категория сигнала (PREMIUM/STANDARD/RISKY/WEAK)
- `btc_trend_strength` - сила тренда BTC (STRONG_BULL/BULL/NEUTRAL/BEAR/STRONG_BEAR)

### 2. Категории сигналов

Определяются автоматически на основе силы сигнала:
- **PREMIUM** (75%+) - самые сильные сигналы
- **STANDARD** (65-74%) - стандартные сигналы
- **RISKY** (55-64%) - рискованные сигналы
- **WEAK** (<55%) - слабые сигналы

### 3. Сила тренда BTC

Определяется на основе изменения BTC за 12ч (4h таймфрейм):
- **STRONG_BULL** (+5%+) - сильный рост
- **BULL** (+3% до +5%) - рост
- **NEUTRAL** (-3% до +3%) - нейтрально
- **BEAR** (-5% до -3%) - падение
- **STRONG_BEAR** (-5%+) - сильное падение

### 4. Telegram уведомления

Теперь показывают:
```
💎 PREMIUM
BTC: 🚀🚀 STRONG_BULL
```

### 5. Скрипт анализа

`analyze_categories.py` - показывает статистику:
- Винрейт по категориям
- Винрейт по силе тренда BTC и направлению
- Винрейт по сессиям
- Лучшие комбинации (категория + BTC тренд + направление)
- Рекомендации по фильтрам

## Как использовать

### Шаг 1: Сбор данных (2-3 дня)

1. Включи AUTO_TAKE:
```bash
# В Telegram боте нажми кнопку "🤖 Авто-вход: ВЫКЛ"
# Или в .env установи:
AUTO_TAKE_SIGNALS=true
```

2. Установи минимальную силу сигнала на 50% чтобы собрать больше данных:
```bash
# В .env:
MIN_SIGNAL_STRENGTH=50
```

3. Дай боту поработать 2-3 дня, собрать 200-300 сделок

### Шаг 2: Анализ данных

Запусти скрипт анализа:

### Шаг 3: Включи фильтры

На основе статистики добавь фильтры в `config.py` или `.env`:

```python
# Пример: если PREMIUM показал 62% винрейт, а RISKY 38%
SIGNAL_CATEGORY_FILTER = ['PREMIUM', 'STANDARD']  # только эти категории

# Если LONG на медвежьем рынке показал 28% винрейт
BLOCK_LONG_ON_BEAR = True

# Если SHORT на бычьем рынке показал 24% винрейт
BLOCK_SHORT_ON_BULL = True

# Если азиатская сессия показала 38% винрейт
BLOCK_ASIAN_SESSION = True
```

## Что записывается

Каждый сигнал записывается с полным контекстом:
- Категория сигнала
- Сила тренда BTC
- Направление (LONG/SHORT)
- Сессия (European/American/Asian)
- Все индикаторы
- Исход (TP1/TP2/SL)

Это позволяет анализировать:
- Какие категории реально работают
- Как BTC влияет на винрейт
- В какое время лучше торговать
- Какие комбинации дают лучший результат

## Преимущества подхода

1. **Факты вместо гаданий** - собираем реальные данные, а не гадаем
2. **Адаптивность** - фильтры основаны на твоих данных, не на чужих
3. **Прозрачность** - видишь что работает, что нет
4. **Улучшение** - можешь постоянно анализировать и улучшать фильтры

## Следующие шаги

После 2-3 дней сбора данных:
1. Запусти `python analyze_categories.py`
2. Посмотри какие категории работают
3. Включи фильтры для лучших категорий
4. Продолжай собирать данные с фильтрами
5. Через неделю снова проанализируй - улучшился ли винрейт

Цель: найти комбинации с винрейтом 55%+ и торговать только их.
