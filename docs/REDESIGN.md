# 🎨 Редизайн сайта Futures Terminal

## ✅ Что сделано

### 1. **Полностью новый дизайн**
- **Современная тёмная тема** в стиле GitHub Dark
- **Чистая цветовая палитра**: основные цвета - синий, зелёный, красный
- **Профессиональные шрифты**: Inter + JetBrains Mono
- **Плавные анимации** и переходы
- **Адаптивный дизайн** для мобильных устройств

### 2. **Улучшенная структура**

#### Вкладки навигации:
- 📡 **Сигналы** - активные сигналы с фильтрами
- 📊 **Позиции** - открытые позиции с P&L в реальном времени
- ⏳ **Лимитки** - ожидающие исполнения ордера
- 📈 **История P&L** - закрытые сделки + график доходности + heatmap
- 📉 **Статистика** - подробная статистика торговли

### 3. **Новые функции**

#### Фильтры сигналов:
- Фильтр по направлению (LONG/SHORT/Все)
- Ползунок минимальной силы сигнала
- Счётчик найденных сигналов

#### Графики:
- **BTC график** с таймфреймами (1m, 5m, 15m, 1h, 4h)
- **Кривая доходности** (P&L chart)
- **Heatmap P&L по монетам**

#### Уведомления:
- Всплывающие уведомления о действиях (Взять/Пропустить/Закрыть)
- Анимированные уведомления справа сверху

#### Fear & Greed:
- Визуальный индикатор с градиентом
- Эмодзи и описание
- Таймер до следующего funding

### 4. **API Endpoints** (server.py)

```python
POST /api/take-signal?signal_id={id}     # Взять сигнал
POST /api/skip-signal?signal_id={id}     # Пропустить сигнал
POST /api/close-position?trade_id={id}   # Закрыть позицию
GET  /api/candles/{symbol}               # Получить свечи для графика
```

### 5. **Мобильная версия**
- Адаптивная сетка карточек
- Трансформация таблиц в карточки на мобильных
- Горизонтальная прокрутка вкладок
- Оптимизированные шрифты

## 🚀 Как использовать

### 1. Запустить сервер:
```bash
python server.py
# или
python -m uvicorn server:app --host 0.0.0.0 --port 8000 --reload
```

### 2. Открыть в браузере:
```
http://localhost:8000
```

### 3. WebSocket подключится автоматически:
- Real-time обновления данных
- Reconnect при обрыве соединения

## 🔧 Настройки

### В index.html можно изменить:
- `MAX_RECONNECT_ATTEMPTS` - макс. попыток переподключения WebSocket
- `updateFundingCountdown()` - интервал обновления funding таймера

### В server.py можно изменить:
- Порт сервера (сейчас 8000)
- Интервал push обновлений (сейчас 30 сек)
